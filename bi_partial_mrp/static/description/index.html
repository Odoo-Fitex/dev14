Process Partially finish Manufacturing order Odoo Apps

This Odoo apps helps to you to process partially/semi/half completed product order for goods in MRP production in Odoo. When you start the MO in odoo then default Odoo doesn't have facility to mark as done with partial finish order you can not finished the Manufacturing order untill you produce all the planned quantity of MO. This Odoo apps helps to complete production order with partial finished goods, you can make manufacturing order to done without finish all planned quantity. Sometimes in real scenario if some raw material get damaged or corrupt then you finished half of the order in this situation there is no back order in odoo to finish Manufacturing order with partial quantity . Our this module helps to do that , after installing this odoo apps you can finish Manufacturing order at anytime with partial or semi finished production order.

Features

User can partially done Manufacturing Order.

When user does not have enough quantity to produce MO products and he wants to done the MO with available quantity through this module, user can done the MO.

Done Quantity Same as Available Quantity.

In stock moves deduction it will consider available quantity of consumed product.

Manufacturing Order

On manufacturing order you can see a button for proceed the Manfacturing order partially.

Wizard for entering quantity

After clicking on button you will have wizard where you have to enter quantity less then Quantity To Produce the Manfacturing order with it's lot number if you selected the product track by lots or serial numbers.

After producing Manfacturing order with less quantity the consumed quantity will be same as partial quantity but in finished product it will show you to be produced quantity same as define on Manfufacturing order.
